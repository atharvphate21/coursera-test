HTML stands for Hypertext Markup language. HTML is used to creat web pages and web application.It is widely used language on web. Wecan creat static website by HTML only. Technically,HTML is markup language rather than programming language.

CSS
CSS stands for Cascading Style Sheet. CSS is used to design HTML pages. CSS is widely used language on the web. It helps the web designer to apply style on HTML tags.

JavaScript is an object oriented programming language tht uses prototypes rather than using classes for inheritance. It is light weighted and interpreted language. It is case-sensitive language. It provides good control to user over the web browsers.
